# Process graduation data for a specific level

Process graduation data for a specific level

## Usage

``` r
process_grad_level(df, class_year, level_type)
```

## Arguments

- df:

  Raw data frame for a level (campus/district/county)

- class_year:

  Class year

- level_type:

  "Campus", "District", or "County"

## Value

Processed data frame
