# Download a single AEIS data file

Download a single AEIS data file

## Usage

``` r
download_aeis_file(end_year, dsname, keys, id_param)
```

## Arguments

- end_year:

  School year end

- dsname:

  Dataset name (cstud, dstud, cref, dref)

- keys:

  Vector of key names to request

- id_param:

  ID parameter string (e.g., "dist0=999999")

## Value

Data frame
