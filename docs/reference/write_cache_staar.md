# Write STAAR cache

Write STAAR cache

## Usage

``` r
write_cache_staar(data, year, level, grade, subject, cache_type)
```

## Arguments

- data:

  Data frame to cache

- year:

  Assessment year

- level:

  "district" or "campus"

- grade:

  Grade level

- subject:

  "english" or "spanish"

- cache_type:

  "tidy" or "processed"
