# Check if STAAR cache exists

Check if STAAR cache exists

## Usage

``` r
cache_exists_staar(year, level, grade, subject, cache_type)
```

## Arguments

- year:

  Assessment year

- level:

  "district" or "campus"

- grade:

  Grade level

- subject:

  "english" or "spanish"

- cache_type:

  "tidy" or "processed"

## Value

Logical indicating if cache exists
