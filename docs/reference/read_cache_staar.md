# Read STAAR cache

Read STAAR cache

## Usage

``` r
read_cache_staar(year, level, grade, subject, cache_type)
```

## Arguments

- year:

  Assessment year

- level:

  "district" or "campus"

- grade:

  Grade level

- subject:

  "english" or "spanish"

- cache_type:

  "tidy" or "processed"

## Value

Cached data frame or NULL if cache doesn't exist
