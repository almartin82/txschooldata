# Build TAPR download URL

Constructs the download URL for a TAPR data file based on year and file
type.

## Usage

``` r
build_tapr_url(end_year, file_type)
```

## Arguments

- end_year:

  School year end

- file_type:

  One of "cref", "dref", "cstud", "dstud", "sref", "sstud"

## Value

URL string
