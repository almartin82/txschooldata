# Get STAAR cache path

Generates cache file path for STAAR data.

## Usage

``` r
get_staar_cache_path(year, level, grade, subject, cache_type)
```

## Arguments

- year:

  Assessment year

- level:

  "district" or "campus"

- grade:

  Grade level

- subject:

  "english" or "spanish"

- cache_type:

  "tidy" or "processed"

## Value

File path for cache
